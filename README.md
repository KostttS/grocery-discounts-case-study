# Grocery Discounts — Grocery Promotion Search App

Mobile app concept for searching grocery discounts, promotions and store offers by categories.

The project focused on helping users find discounted products, compare offers, build a basket and estimate store distance.

## Main workflows

### Promotion search

Users can browse or search discounted products and filter them by category or store.

### Basket calculation

The app allows users to add discounted products to a basket and calculate the total value of selected items.

### Store distance

The app structure includes store location data and distance logic so users can understand which offers are practical to visit.

---

## Technical challenges

### Promotion data structure

The app required a clear structure for products, stores, categories, prices, discounts and active promotions.

### Basket logic

The basket flow needed to calculate totals based on selected products, quantities and current promotional prices.

### Practical mobile UX

The main challenge was to keep the app simple: users should quickly find useful discounts without going through a complicated marketplace-style flow.
